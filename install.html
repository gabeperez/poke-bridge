Setup Guide

Connect Poke
to VoiceOS

Control Poke by voice on your Mac. Four steps, five minutes.

Step 01

Get your Poke API key

  1. Open poke.com/kitchen in your browser
  2. Go to API Keys
  3. Create a new V2 key
  4. Copy the key — you'll need it in Step 3
Important

You need a V2 key, not a V1 key. V1 keys start with pk_ and won't work.

Step 02

Download poke-bridge

Open Terminal on your Mac and paste these commands. If you already have the folder, just cd poke-bridge instead.

git clone https://github.com/gabeperez/poke-bridge.git
cd poke-bridge
npm install
Tip

You can open Terminal from Spotlight — press Cmd + Space and type "Terminal".

Step 03

Add your API key

Still in Terminal, run:

cp start.sh.example start.sh
chmod +x start.sh

Now open start.sh in any text editor and replace your_v2_key_here with the API key you copied in Step 1.

# start.sh should look like this:

export POKE_API_KEY="paste_your_key_here"
export POKE_SOURCE="VoiceOS"

Save and close the file.

Step 04

Add to VoiceOS

Open VoiceOS and navigate to:

Settings → Integrations → Custom Integrations → Add

Fill in the fields:

Name Poke
Launch Command /full/path/to/poke-bridge/start.sh
Finding the path

In Terminal, type cd poke-bridge && pwd to see the full path. Add /start.sh to the end.

You're set

Start talking

Speak to VoiceOS naturally. Mention Poke and it will route your request.

Ask Poke to remind me to call the client tomorrow at 9am
Tell Poke to draft an email to the team about Friday's deadline
Have Poke add a standup to my calendar Monday at 10am
Ask Poke what's on my calendar today

Poke will reply in iMessage with the result.
