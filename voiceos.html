Setup Guide セットアップガイド

Connect Poke
to VoiceOS

Control Poke by voice on your Mac or PC. Four steps, five minutes. MacまたはPCで声を使ってPokeを操作しよう。4つのステップ、5分で完了。

Step 01

Get your Poke API key PokeのAPIキーを取得する

  1. Open poke.com/kitchen in your browser
  2. Go to API Keys
  3. Create a new V2 key
  4. Copy the key — you'll need it in Step 3
  1. ブラウザで poke.com/kitchen を開く
  2. API Keys をクリック
  3. 新しい V2 キーを作る
  4. キーをコピーする — ステップ3で使います
Important 注意

You need a V2 key, not a V1 key. V1 keys start with pk_ and won't work. V1キーではなくV2キーが必要です。V1キーは pk_ で始まりますが、使えません。

Step 02

Download poke-bridge poke-bridgeをダウンロードする

Open a terminal and paste this command. It will create a new folder and set everything up for you. ターミナルを開いて、このコマンドを貼り付けてください。フォルダが自動で作られて、準備が完了します。

git clone https://github.com/gabeperez/poke-bridge.git && cd poke-bridge && npm install
Tip

macOS: Open Terminal from Spotlight — press Cmd + Space and type "Terminal".
Windows: Open Command Prompt or PowerShell — press Win + R, type cmd, and press Enter. macOS: Spotlightからターミナルを開けます — Cmd + Space を押して「Terminal」と入力。
Windows: コマンドプロンプトまたはPowerShellを開きます — Win + R を押して cmd と入力してEnter。

Step 03

Add your API key APIキーを設定する

macOS / Linux:

Still in your terminal, run: ターミナルで続けて、これを実行してください：

cp start.sh.example start.sh
chmod +x start.sh

Open start.sh in any text editor and replace your_v2_key_here with the API key you copied in Step 1. start.sh をテキストエディタで開いて、your_v2_key_here をステップ1でコピーしたAPIキーに置き換えてください。

# start.sh should look like this:# start.sh はこんな感じになります：

export POKE_API_KEY="paste_your_key_here"
export POKE_SOURCE="VoiceOS"

Windows:

In Command Prompt or PowerShell, run: コマンドプロンプトまたはPowerShellで、これを実行してください：

copy start.cmd.example start.cmd

Open start.cmd in any text editor and replace your_v2_key_here with the API key you copied in Step 1. start.cmd をテキストエディタで開いて、your_v2_key_here をステップ1でコピーしたAPIキーに置き換えてください。

REM start.cmd should look like this:REM start.cmd はこんな感じになります：

set POKE_API_KEY=paste_your_key_here
set POKE_SOURCE=VoiceOS

Save and close the file. ファイルを保存して閉じてください。

Step 04

Add to VoiceOS VoiceOSに追加する

Open VoiceOS and navigate to: VoiceOS を開いて、ここに進んでください：

Settings → Integrations → Custom Integrations → Add

Fill in the fields: 以下の内容を入力してください：

Name Poke
Launch Command (macOS) 起動コマンド（macOS） /full/path/to/poke-bridge/start.sh
Launch Command (Windows) 起動コマンド（Windows） C:\full\path\to\poke-bridge\start.cmd
Finding the path パスの確認方法

macOS: In Terminal, type cd poke-bridge && pwd to see the full path. Add /start.sh to the end.
Windows: In Command Prompt, type cd poke-bridge && cd to see the full path. Add \start.cmd to the end. macOS: ターミナルで cd poke-bridge && pwd と入力するとパスが表示されます。その後ろに /start.sh を付けてください。
Windows: コマンドプロンプトで cd poke-bridge && cd と入力するとパスが表示されます。その後ろに \start.cmd を付けてください。

You're set 準備完了

Start talking 話しかけてみよう

Speak to VoiceOS naturally. Mention Poke and it will route your request. VoiceOSに自然に話しかけてください。Pokeの名前を言えば、リクエストが届きます。

Ask Poke to remind me to call the client tomorrow at 9am
Tell Poke to draft an email to the team about Friday's deadline
Have Poke add a standup to my calendar Monday at 10am
Ask Poke what's on my calendar today
Pokeに明日の朝9時にクライアントに電話するリマインダーをお願いして
Pokeに金曜の締め切りについてチームにメールを書いてもらって
Pokeに月曜の朝10時にスタンドアップをカレンダーに追加して
Pokeに今日のカレンダーを聞いて

Poke will reply in iMessage with the result. Pokeの返事はiMessageに届きます。
